## Sistema de Combate em JavaScript

Projeto desenvolvido com HTML, CSS e JavaScript puro, simulando um sistema de combate por turnos aleatórios.

### Conceitos aplicados: 
- Funções e escopo;
- Manipulação de DOM;
- Condicionais e lógica de jogo;
- Gerenciamento de estado;
- Eventos;
- Barra de vida dinâmica;
- Feedback visual;
- Versionamento com Git e Github

## Tecnologias: 
- HTML;
- CSS;
- JavaScript.
